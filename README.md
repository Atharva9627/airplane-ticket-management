# ✈️ Airplane Ticket Management System

A full-stack web application for managing airline bookings. This project features a modern, responsive frontend and a robust Python-based backend connected to a PostgreSQL database.

## 🚀 Features
- **Book Tickets:** Add new flight bookings with date, class, and fare.
- **View Records:** Real-time table display of all booked tickets.
- **Update Bookings:** Modify class or fare for existing tickets using their ID.
- **Cancel Tickets:** Delete records from the database.
- **Responsive UI:** Clean, dark-themed dashboard with "Toast" notifications.

## 🛠️ Tech Stack
- **Frontend:** HTML5, CSS3 (Flexbox/Grid), JavaScript (Fetch API)
- **Backend:** Python, Flask, Flask-CORS
- **Database:** PostgreSQL
- **Driver:** Psycopg2
